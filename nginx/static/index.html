Key Features

Plagiarism Detection

Uses AI to compare student code submissions against a comprehensive database of solutions.

Dynamic Reporting

Provides instructors with dynamic reports to track student progress and behavior over time.

Deployment Flexibility

Can be deployed locally or on a server, offering flexible solutions for educational institutions.

Ease of Use

Students submit assignments through a Telegram bot, while instructors analyze results via a web interface.

Target Audience

Educational Institutions

Maintain academic integrity by preventing plagiarism and ensuring transparency in grading.

Instructors

Simplifies plagiarism checking, saving time and effort, and provides dynamic student progress reports.

Students

Encourages independent completion of assignments and provides a record of their work for future reference.

Project Details

StudyFair is an open-source academic integrity tool that detects plagiarism in student programming assignments using AI. It supports multiple programming languages, offers dynamic reporting, and can be deployed flexibly. With a user-friendly interface, it simplifies the submission and analysis process for both students and instructors.
