Studyfair is a free, open-source tool and SaaS solution for automatic code clone detection in academic purposes.

It"s also available as a self-hosted service, see our documentation for installation details.

Code clone detection

This problem has a long history of academic research. Although many tools have been proposed for identifying clones, no system has yet been developed that provides clone detection as a service.

The simplest clone code might look like this: 

          
            def foo(a, b):
              a = a + 1
              b = b + 1

              return a * b
          

          
            def foo(a, b):
              return (a + 1) * (b + 1)

Although in this case the situation is clear to the naked eye, in most cases it is necessary to use advanced algorithms.
